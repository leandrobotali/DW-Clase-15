PASOS A SEGUIR PARA USAR GIT

git --version

git init

git status

git add index.html

git add .

git commit -m "comentario"

git log

git log --oneline

RAMAS

git branch mi_rama

git checkout mi_rama

git checkout + identificador

git branch -D mi_rama

para convinar se posa sobre la rama master y se coloca "git merge mi_rama"

git checkout -b mi_rama2 (crea la rama y te posiciona en esa rama)

cuando hay conflictos entre una rama y el master, luego de decidir con que cambios te quedas hay que hacer uncommit para finalizar el merge

git reset da marcha atras en los commit
